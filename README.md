# Four-storey-building using REVIT ARCHITECTURE AND STRUCTURE

Students are tasked with designing a 4-storey commercial office building using Revit Architecture and Revit Structure , ensuring effective integration between architectural and structural models in a BIM environment. 

Build 4-storey BIM model.

Produce 2D structural + reinforcement drawings.

Add realistic renders, imagewalkthrough for top-quality visuals.
